To run the program:

1. Open your terminal (in Linux)
2. Find the folder 'src'
3. Write: python src_server.py test.py
The server should be running now

4. Open new terminal (in Linux)
5. Find the folder 'src'
6. Write: python src_client.py localhost 8080
The client should be running and the decrpytet/encrypted message should
be on a new txt-file named 'text_client.txt'
